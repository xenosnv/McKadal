# 🦎 McKadal - Food Delivery App

A food delivery application prototype built with Android Studio using Kotlin and XML.

<img width="388" height="839" alt="Screenshot 2026-04-25 054311" src="https://github.com/user-attachments/assets/994b6945-6b6a-4d95-bc75-52d6da0c28c4" />

## 📱 Features
- Login & Welcome screen
- Home with food categories and recommendations
- Food detail page with spicy level selection
- Favorite menu management
- Shopping cart with payment method selection
- Order confirmation and order history
- User profile page

## 🛠️ Tech Stack
- **Language:** Kotlin
- **UI:** XML with ConstraintLayout, CardView, GridLayout
- **Navigation:** Intent-based with slide animations
- **IDE:** Android Studio

## 🚀 How to Run
1. Clone this repository
2. Open with Android Studio
3. Run on emulator or physical device (min SDK 21)
